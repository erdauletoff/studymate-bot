## Ключевые улучшения:

✅ **Медали для топ-3** - 🥇🥈🥉
✅ **Чистый вид** - только балл (без лишней инфо)
✅ **Разделитель** - визуально отделяет топ-3
✅ **Топ-15** - больше мотивации попасть в список
✅ **Моноширинный шрифт** - балл в `<code>` теге
✅ **Позиция вне топа** - красиво оформлена с разделителем
